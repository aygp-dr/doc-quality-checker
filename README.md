# doc-quality-checker

Lint documentation for completeness and accuracy. Built with [Babashka](https://github.com/babashka/babashka).

## Quick Start

```bash
bb run --dir ./my-project
bb run --dir ./my-project --format json
```

## Requirements

- Babashka v1.12+
